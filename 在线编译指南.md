# 在线编译 APK - 无需安装工具

## 方法1：使用 GitHub Actions（推荐）

### 步骤1：上传项目到 GitHub

1. 在 GitHub 创建新仓库
2. 上传整个 `LSPosedAppInstallTime` 文件夹
3. 已包含 `.github/workflows/build.yml`，会自动编译

### 步骤2：触发编译

1. 上传代码后，GitHub Actions 会自动编译
2. 或手动触发：Actions -> Build APK -> Run workflow

### 步骤3：下载 APK

1. 编译完成后，在 Actions 页面
2. 点击最新的 workflow run
3. 在 Artifacts 中下载 `app-debug.apk`

## 方法2：使用 Gitpod（在线 IDE）

### 步骤1：访问 Gitpod

1. 将项目上传到 GitHub
2. 访问：`https://gitpod.io/#你的仓库地址`

### 步骤2：在 Gitpod 中编译

```bash
./gradlew assembleDebug
```

### 步骤3：下载 APK

APK 在：`app/build/outputs/apk/debug/app-debug.apk`

## 方法3：使用 GitHub Codespaces

1. 上传项目到 GitHub
2. 点击 Code -> Codespaces -> Create codespace
3. 在 Codespaces 中编译：
   ```bash
   ./gradlew assembleDebug
   ```
4. 下载 APK
