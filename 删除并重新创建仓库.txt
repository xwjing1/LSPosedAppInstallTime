==========================================
删除 GitHub 仓库并重新开始
==========================================

## 方法1：删除仓库（推荐）

### 步骤1：进入仓库设置

1. 访问：https://github.com/xwjing1/LSPosedAppInstallTime
2. 点击仓库顶部的 "Settings" 标签页
3. 滚动到页面最底部

### 步骤2：删除仓库

1. 找到 "Danger Zone"（危险区域）部分
2. 点击 "Delete this repository"（删除此仓库）
3. 输入仓库名确认：LSPosedAppInstallTime
4. 点击 "I understand the consequences, delete this repository"
5. 确认删除

## 方法2：创建新仓库

### 步骤1：创建新仓库

1. 在 GitHub 首页，点击右上角 "+" -> "New repository"
2. 仓库名：LSPosedAppInstallTime（或其他名字）
3. 选择 Public 或 Private
4. 不要勾选 "Add a README file"（我们要上传完整项目）
5. 点击 "Create repository"

### 步骤2：上传完整项目

1. 在新建的仓库页面，点击 "uploading an existing file"
2. 打开文件夹：C:\Users\Administrator\Downloads\LSPosedAppInstallTime
3. 选择所有文件和文件夹（Ctrl+A）
4. 拖拽到 GitHub 页面
5. 点击 "Commit changes"

### 步骤3：创建 Workflow

1. 点击 "Actions" 标签页
2. 点击 "Skip this and set up a workflow yourself →"
3. 文件名：`.github/workflows/build.yml`
4. 粘贴 workflow 代码（见 workflow代码.txt）
5. 点击 "Start commit" 保存

### 步骤4：运行编译

1. 回到 Actions 页面
2. 点击 "Build APK"
3. 点击 "Run workflow"
4. 等待编译完成
